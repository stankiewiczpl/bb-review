Pomoc

▶ Jak utworzyc API Token?
  1. Wejdz na: id.atlassian.com → Security → API tokens
  2. Kliknij "Create API token"
  3. Zaznacz scope: Repositories: Read, Pull requests: Write
  4. Skopiuj token (zaczyna sie od ATATT3x...)
  5. Wklej w pole "API Token" na ekranie startowym
▶ Jak zainstalowac Claude Code CLI?
  1. Zainstaluj Node.js 18+
  2. npm install -g @anthropic-ai/claude-code
  3. claude auth (jednorazowe logowanie)
  4. Sprawdz: claude --version
▶ Kontekst lokalnego repozytorium

Opcjonalnie podaj sciezke do lokalnego klonu repo. Claude Code uzyje go jako kontekstu — bedzie mogl czytac importy, typy, konwencje projektu.

Sciezka musi wskazywac na katalog z repo, np: /home/user/projekty/moje-repo

Claude NIE modyfikuje plikow — tylko je czyta.

▶ Skad wziac URL pull requesta?

Otworz PR w przegladarce, skopiuj URL z paska adresu:

https://bitbucket.org/{workspace}/{repo}/pull-requests/{numer}

Lub wpisz sam numer PR (wtedy podaj tez workspace i nazwe repo).

▶ Logi i historia

Kazde review jest automatycznie zapisywane w katalogu logs/ jako plik JSON.

Przegladaj historie klikajac Logi w naglowku.

Jesli PR byl juz reviewowany, zobaczysz ostrzezenie z data ostatniego review.
